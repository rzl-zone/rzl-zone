## Supported Versions

Only the latest stable release of **UtilsJS** will receive security updates.  
Please make sure you are using the most recent version before reporting an issue.

<SeparatorSection />

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by contacting:

- 📧 Email: rzlzone.dev@gmail.com.
- 🔗 GitHub Issues (private disclosure preferred): [UtilsJS Repository](https://github.com/rzl-zone/utils-js).

We will review and respond to your report as quickly as possible.  
Please **do not disclose the vulnerability publicly** until it has been addressed.

<SeparatorSection />

## Responsible Disclosure

To protect the community, we kindly ask that you:

1. Provide detailed steps to reproduce the vulnerability.
2. Allow reasonable time for us to investigate and patch before any public disclosure.
3. Avoid exploiting vulnerabilities for any malicious purpose.

<SeparatorSection />

## Scope

- ✅ Vulnerabilities affecting the core logic or data integrity of UtilsJS.
- ❌ Non-security bugs (use the [issue tracker](https://github.com/rzl-zone/utils-js/issues) for those).

<SeparatorSection />

🙏 Thank you for helping keep **UtilsJS** and its users safe!
